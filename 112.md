### git bash
- 명령어
	- ./ : 현재 폴더
	- ../ :현재 작업하고 있는 폴더의 부모 폴더
	- cd : change directory
	- ls : list segment(현재 작업중인 디렉토리의 폴더/ 파일 목록)
	- mkdir : make directory
	- mv : move
	- rm : remove
